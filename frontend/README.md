# ScenGen Frontend

React frontend for the CARLA Scenario Generator.

## Features

- **Dashboard** — Overview of generated scenarios
- **Scenario Builder** — Manual scenario configuration
- **AI Generator** — Natural language scenario generation
- **Batch Generator** — Generate multiple scenarios at once
- **Library** — Browse and manage scenarios
- **Simulator** — Run scenarios in CARLA with live preview

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
